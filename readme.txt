PlusWords shows the most used words on the entire blog and adds a link to search them

== Description ==

PlusWords shows through a flash applet the most used words in the entire blog randomly positioning them also by size and adds a link to search the word
There's the option to configure the number of minimum occurrences and lenght of the words to be displayed.

== Installation ==

1. Create a pluswords directory on "/wp-content/plugins/pluswords" and upload to it the files "exclude.txt" and pluswords.php
1. Put the "pluswords.swf" on the root of your site "/"
1. Activate the plugin through the 'Plugins' menu in WordPress
